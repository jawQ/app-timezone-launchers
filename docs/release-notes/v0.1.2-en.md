> This release focuses on **shipping and install experience**: English-default release notes with a full Chinese notes entry, one-command maintainer tagging via `pnpm release:tag`, and screenshot-backed Gatekeeper guidance for first open.

---

## Overview

ZoneLaunch v0.1.2 is a maintenance release after v0.1.1. Core time-zone launch behavior is unchanged; work centers on documentation and the release pipeline.

You can still keep the system time zone for work while launching WeChat, Feishu/Lark, and other apps under a chosen `TZ` (or the reverse).

---

## Highlights

- **Two language packs per release**: English on the GitHub Release page by default; full Chinese notes via **中文** entry (`RELEASE_NOTES.zh-CN.md` / `docs/release-notes/v*-zh.md`).
- **One-command release**: `pnpm release:tag` (same idea as supermarkets `miniapp:tag`) — tag push triggers CI for an ad-hoc signed zip.
- **Gatekeeper docs + screenshots**: README and install docs cover **Not Opened** and **Open Anyway**.
- **Background copy**: documents the “system TZ vs physical location vs social apps” pain point.
