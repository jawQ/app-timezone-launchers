## Overview

ZoneLaunch v0.1.2 is a maintenance release after v0.1.1. Core time-zone launch behavior is unchanged; work centers on documentation and the release pipeline.

You can still keep the system time zone for work while launching WeChat, Feishu/Lark, and other apps under a chosen `TZ` (or the reverse).

---

## Highlights

- **Bilingual release notes**: Chinese-first GitHub Release body (overview / highlights / details) with a top link to full English notes; curated files under `docs/release-notes/` or auto from git.
- **One-command release**: root `package.json` scripts (`pnpm release:tag`), same idea as supermarkets `miniapp:tag` — tag push triggers CI for an ad-hoc signed zip.
- **Gatekeeper docs + screenshots**: README and install docs cover **Not Opened** and **Open Anyway** so users do not trash the app by mistake.
- **Background copy**: documents the “system TZ vs physical location vs social apps” pain point in EN/ZH.

## Download & install

**Assets**

- `ZoneLaunch-0.1.2-macos.zip` — `ZoneLaunch.app` + `README-FIRST.txt` (ad-hoc signed, **not notarized**)
- `SHA256SUMS` — checksum

**Install**

1. Unzip and drag **ZoneLaunch.app** into Applications.
2. First open may be blocked: **Done**, then **System Settings → Privacy & Security → Open Anyway**.
3. Screenshots: [Install from Releases](https://github.com/jawQ/app-timezone-launchers/blob/master/docs/app/install-from-release.md)

**Shell launchers (lighter)**

```bash
git clone https://github.com/jawQ/app-timezone-launchers.git
cd app-timezone-launchers
./install.sh --feishu --wechat
```

**Identity:** `app.zonelaunch.launcher`
