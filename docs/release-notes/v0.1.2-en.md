> This release focuses on **shipping and install experience**: bilingual GitHub Release notes (English default, Chinese entry), one-command maintainer tagging via `pnpm release:tag`, and screenshot-backed Gatekeeper guidance for first open.

---

## Overview

ZoneLaunch v0.1.2 is a maintenance release after v0.1.1. Core time-zone launch behavior is unchanged; work centers on documentation and the release pipeline.

You can still keep the system time zone for work while launching WeChat, Feishu/Lark, and other apps under a chosen `TZ` (or the reverse).

---

## Highlights

- **Bilingual release notes**: **English** is the default GitHub Release body; **[中文 →](v0.1.2-zh.md)** entry links to full Chinese notes. Curated files under `docs/release-notes/` or auto from git.
- **One-command release**: root `package.json` scripts (`pnpm release:tag`), same idea as supermarkets `miniapp:tag` — tag push triggers CI for an ad-hoc signed zip.
- **Gatekeeper docs + screenshots**: README and install docs cover **Not Opened** and **Open Anyway** so users do not trash the app by mistake.
- **Background copy**: documents the “system TZ vs physical location vs social apps” pain point in EN/ZH.
