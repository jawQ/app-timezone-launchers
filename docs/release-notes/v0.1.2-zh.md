## 概览

ZoneLaunch v0.1.2 是 v0.1.1 之后的维护更新，主要面向**文档与发版链路**，不改变按时区启动应用的核心机制。

系统时区与社交软件时间不一致时，仍可通过脚本或 ZoneLaunch App 为指定应用注入 `TZ`；本版让「下载 App → 首次打开 → 发下一版」的路径更清晰。

---

## 重点内容

- **每版双语文档**：英文为 GitHub Release 默认正文；中文为独立完整说明（`RELEASE_NOTES.zh-CN.md` / `docs/release-notes/v*-zh.md`），互不夹带。
- **维护者一键发版**：`pnpm release:tag`（对齐 supermarkets 的 `miniapp:tag`），自动 patch 递增、推送 tag，由 Actions 打包 ad-hoc 签名 zip。
- **门禁安装说明 + 截图**：说明「Not Opened」与 **Privacy & Security → Open Anyway**，避免误点移到废纸篓。
- **项目背景文案**：补充「改系统时区导致微信 / 飞书时间与物理时区不一致」的痛点与方案。

## 下载与安装

**资源**

- `ZoneLaunch-0.1.2-macos.zip` — `ZoneLaunch.app` + `README-FIRST.txt`（ad-hoc 签名，**未公证**）
- `SHA256SUMS` — 校验和
- `RELEASE_NOTES.md` — 英文更新说明（GitHub Release 默认展示）

**安装步骤**

1. 解压后将 **ZoneLaunch.app** 拖到「应用程序」。
2. 首次打开常被门禁拦截：点 **Done / 完成**，再到 **系统设置 → 隐私与安全性 → Open Anyway / 仍要打开**。
3. 图文说明：[从 Release 安装](https://github.com/jawQ/app-timezone-launchers/blob/master/docs/app/install-from-release.zh-CN.md)

**更轻量：Shell 启动命令**

```bash
git clone https://github.com/jawQ/app-timezone-launchers.git
cd app-timezone-launchers
./install.sh --feishu --wechat
```

**应用身份：** Bundle ID `app.zonelaunch.launcher`
