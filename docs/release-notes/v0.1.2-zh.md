## 概览

ZoneLaunch v0.1.2 是 v0.1.1 之后的维护更新，主要面向**文档与发版链路**，不改变按时区启动应用的核心机制。

系统时区与社交软件时间不一致时，仍可通过脚本或 ZoneLaunch App 为指定应用注入 `TZ`；本版让「下载 App → 首次打开 → 发下一版」的路径更清晰。

---

## 重点内容

- **中英双语 Release 日志**：Release 页默认中文（概览 / 重点 / 明细），顶部可跳转英文全文；支持手写 `docs/release-notes/v*-zh.md` / `v*-en.md`，或从 git 自动生成。
- **维护者一键发版**：根目录 `package.json` 提供 `pnpm release:tag`（对齐 supermarkets 的 `miniapp:tag`），自动 patch 递增、推送 tag，由 Actions 打包 ad-hoc 签名 zip。
- **门禁安装说明 + 截图**：主 README 与安装文档说明「Not Opened」与 **Privacy & Security → Open Anyway**，避免用户误点移到废纸篓。
- **项目背景文案**：补充「改系统时区导致微信 / 飞书时间与物理时区不一致」的痛点与方案说明（中英）。

## 变更
