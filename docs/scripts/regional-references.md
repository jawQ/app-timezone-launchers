# Regional references

[Simplified Chinese](regional-references.zh-CN.md)

Representative office and chat apps by region. Use these when choosing a time zone; they are not strict rankings.

> macOS only. This project targets macOS `.app` launchers. Windows and Linux are unsupported.

## Reference table

| Region | Time zone | Common work tools | Common chat tools |
| --- | --- | --- | --- |
| San Francisco, US | `America/Los_Angeles` | Microsoft Teams, Slack, Zoom | Messenger, iMessage, WhatsApp |
| Germany, Europe | `Europe/Berlin` | Microsoft Teams, Slack, Zoom | WhatsApp, Telegram, Signal |
| Japan | `Asia/Tokyo` | Microsoft Teams, LINE WORKS, Chatwork | LINE, WhatsApp, Messenger |
| South Korea | `Asia/Seoul` | Microsoft Teams, Slack, KakaoWork | KakaoTalk, Telegram, WhatsApp |
| Singapore | `Asia/Singapore` | Microsoft Teams, Slack, Zoom | WhatsApp, Telegram, Messenger |

## Feishu / Lark examples

```bash
LARK_TZ=America/Los_Angeles feishu-tz
LARK_TZ=Europe/Berlin feishu-tz
LARK_TZ=Asia/Tokyo feishu-tz
LARK_TZ=Asia/Seoul feishu-tz
LARK_TZ=Asia/Singapore feishu-tz
```
